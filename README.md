# P-wave-arrival-time-picking

Data Availability
The data used in this study are obtained from the STanford EArthquake Dataset (STEAD).

Source: https://github.com/smousavi05/STEAD

Reference:
Mousavi, S. M., Sheng, Y., Zhu, W., & Beroza, G. C. (2019). STanford EArthquake Dataset (STEAD): A Global Data Set of Seismic Signals for AI. IEEE Access, 7, 179464-179476.

For data access and usage instructions, please visit the STEAD GitHub repository.
